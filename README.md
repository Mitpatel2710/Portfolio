# Mit Portfolio

## Project Structure

The app lives inside the `Mit-Portfolio/` subdirectory.

## Setup & Running

```bash
# Navigate to the app directory
cd Mit-Portfolio

# Install dependencies
npm install

# Start the development server (http://localhost:5173)
npm run dev
```

## Build for Production

```bash
cd Mit-Portfolio
npm run build
```

The production-ready files will be output to `Mit-Portfolio/dist/`.

## Preview Production Build

```bash
cd Mit-Portfolio
npm run preview
```
